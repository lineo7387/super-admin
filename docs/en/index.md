## What This Is

Super Admin is an open-source admin template focused on the frontend experience first. The primary public path is creating your own admin project with `npm create super-admin@latest`, then replacing the example data and modules with your business workflow.

The default starter stays mock-backed on purpose. Backend, database, auth, AI, and formal API contracts remain optional integration surfaces.

## Which Path Should I Use?

| Path                    | Audience                                                                      | Start Here                                            |
| ----------------------- | ----------------------------------------------------------------------------- | ----------------------------------------------------- |
| Use Super Admin         | Teams creating their own admin app from the npm starter.                      | [Getting Started](guide/getting-started.md)           |
| Connect your API        | Users replacing mock data with their own backend responses.                   | [API Adapters](guide/api-adapters.md)                 |
| Develop this repository | Maintainers changing the Super Admin source, packages, docs, or release flow. | [Open Source Workflow](guide/open-source-workflow.md) |

Maintainer tools such as Trellis, Codex/Claude settings, release scripts, and GitHub presentation checklists belong to the source repository. They are not required in projects generated by `create-super-admin`.
